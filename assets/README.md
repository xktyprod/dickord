# Icon Instructions

To change the application icon:

1. Create or download an icon file in .ico format
   - Recommended size: 256x256 pixels
   - Can include multiple sizes: 16x16, 32x32, 48x48, 256x256

2. Name it `icon.ico`

3. Place it in this `assets` folder

4. Rebuild the application using `build.bat`

## Online Icon Converters:
- https://convertio.co/ru/png-ico/
- https://www.icoconverter.com/
- https://favicon.io/favicon-converter/

## Current Configuration:
The package.json is already configured to use `assets/icon.ico`
