# Sound Effects

This directory contains sound effect files for the Discord Clone application.

## Required Sound Files

You need to add the following sound files to this directory:

1. **voice-join.mp3** - Played when joining a voice channel
2. **voice-leave.mp3** - Played when leaving a voice channel
3. **mic-toggle.mp3** - Played when toggling microphone on/off
4. **incoming-call.mp3** - Played for incoming calls (ringtone)
5. **message-send.mp3** - Played when sending a message
6. **mention.mp3** - Played when receiving a mention

## Sound Requirements

- Format: MP3 or OGG
- Duration: 0.5-2 seconds recommended
- Volume: Normalized to prevent clipping

## Free Sound Resources

You can find free sound effects at:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/
